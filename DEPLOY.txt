📋 БЫСТРАЯ ИНСТРУКЦИЯ:

1. Зайдите на github.com и создайте репозиторий
   └─ Название: techstore (или любое)
   └─ Тип: Public

2. Загрузите ВСЕ файлы проекта:
   ├─ index.html
   ├─ styles.css
   ├─ script.js
   ├─ images/ (вся папка)
   ├─ models/ (вся папка)
   └─ остальные файлы

3. Включите GitHub Pages:
   Settings → Pages → Branch: main → Save

4. Через 1-2 минуты сайт готов:
   https://ВАШ_НИК.github.io/techstore/
