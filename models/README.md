# 3D Модели

Эта папка предназначена для хранения FBX моделей товаров.

## Как добавить 3D модель

1. Поместите FBX файл в эту папку
2. Обновите путь `model3D` в массиве `products` в файле `script.js`

## Пример

```
models/
├── macbook.fbx
├── iphone.fbx
├── ipad.fbx
└── xbox.fbx
```

В `script.js`:
```javascript
{
    id: 1,
    name: "MacBook Pro 16\"",
    has3D: true,
    model3D: "models/macbook.fbx"  // ← Укажите путь к файлу
}
```

## Совместимость

- **Формат:** FBX
- **Размер:** Рекомендуется < 10 MB
- **Оптимизация:** Используйте Blender для оптимизации моделей

## Где найти модели

Вы можете найти FBX модели на:
- Sketchfab (многие модели доступны бесплатно)
- TurboSquid
- CGTrader
- Free3D

## Конвертация моделей

Если у вас модель в другом формате (GLB, GLTF, OBJ):
1. Используйте Blender
2. Импортируйте модель
3. Экспортируйте в FBX формат
