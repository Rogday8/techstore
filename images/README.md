# Папка с изображениями товаров

## Как добавить свои фотографии

1. Скачайте PNG/JPG изображения ваших товаров
2. Поместите их в эту папку `images/`
3. В файле `script.js` обновите пути к изображениям

## Рекомендуемые размеры

- **Главное изображение товара:** 400x280 пикселей
- **Изображения для галереи:** 800x600 пикселей
- **Формат:** PNG или JPG

## Пример использования

```javascript
{
    id: 1,
    name: "iPhone 15 Pro Max",
    image: "images/iphone15.jpg",  // ← Ваше изображение
    images: [
        "images/iphone15-1.jpg",
        "images/iphone15-2.jpg",
        "images/iphone15-3.jpg"
    ]
}
```

## Где найти изображения

- **Icons8** - бесплатные изображения товаров
- **Unsplash** - бесплатные стоковые фото
- **Pexels** - бесплатные фото высокого качества
- **Pixabay** - бесплатные изображения

## Где скачать реальные фото техники

- **Apple** - официальный сайт apple.com
- **Samsung** - официальный сайт samsung.com
- **Amazon** - там есть фото товаров
- **AliExpress** - фото реальных товаров
